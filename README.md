## 🚀 Getting Started (Local Development)

### 1️⃣ Clone the repository

```bash
git clone https://github.com/kumar-shubham1/rajinderdadhaba.git
cd rajinderdadhaba
````

### 2️⃣ Install dependencies

```bash
npm install
```

or

```bash
yarn install
```

### 3️⃣ Run development server

```bash
npm run dev
```

Open:

```
http://localhost:3000
```

---

## 📦 Build for Production

```bash
npm run build
npm start
```

## 📁 Project Structure

```
/public        → Images & static assets

  /pages       → Site pages (Next.js) or routes
  /styles      → Global styles
  /utils       → Helper functions
```
